Wat is dit?

Welkom op de documentatie van de Magic Mirror. Hier vind je alle informatie die je nodig hebt om met de Magic Mirror aan de slag te gaan.

Aan de slag

Om aan de slag te gaan met dit project, is het vanzelfsprekend handig om het project lokaal te hebben staan. Het makkelijkst is om het project binnen te halen via de terminal:

$ cd jouw_repository_root # Vul hier de map in, waar je het project op wilt slaan.
$ git clone https://github.com/deltafhict/mirror-docs.git

Je kunt natuurlijk, wanneer je bij de organisatie aangesloten bent, gebruik maken van de Github-client (beschikbaar voor OS X & Windows) om de repository te clonen.
